# Omron FINS Protocol MCP Server (NPM)

A Node.js/TypeScript implementation of the Model Context Protocol (MCP) server for Omron FINS Protocol.

## Installation

```bash
npm install
```

## Usage

```bash
npm run build
npm start
```

## Configuration

Create a `.env` file with the following variables:

```bash
FINS_HOST=192.168.1.10
FINS_PORT=9600
FINS_PROTOCOL=tcp
```
