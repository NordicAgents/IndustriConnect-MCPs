# Beckhoff ADS Protocol MCP Server (Python)

A Python implementation of the Model Context Protocol (MCP) server for Beckhoff ADS Protocol.

## Installation

```bash
uv sync
```

## Usage

```bash
uv run ads-mcp
```

## Configuration

Create a `.env` file with the following variables:

```bash
ADS_NET_ID=127.0.0.1.1.1
ADS_PORT=851
```
