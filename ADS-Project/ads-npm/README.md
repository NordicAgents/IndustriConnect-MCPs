# Beckhoff ADS Protocol MCP Server (NPM)

A Node.js/TypeScript implementation of the Model Context Protocol (MCP) server for Beckhoff ADS Protocol.

## Installation

```bash
npm install
```

## Usage

```bash
npm run build
npm start
```

## Configuration

Create a `.env` file with the following variables:

```bash
ADS_NET_ID=127.0.0.1.1.1
ADS_PORT=851
```
